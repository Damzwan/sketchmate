## When to Use the Lasso Tool

The Lasso Tool provides precision when you need to select multiple objects, especially in a crowded canvas. It's your
go-to when you want **detailed control** over selection.

## How to Use

Here's a quick step-by-step guide:

1. Select the Lasso Tool.
2. Begin drawing a shape around the objects you wish to select.
3. Release, and you'll automatically switch to Select Mode with all the objects within your lasso selected.
